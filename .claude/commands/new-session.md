Starting a new development session. Before any work:

1. Run: git checkout main && git pull origin main
2. Read CLAUDE.md for project rules and architecture
3. Read MEMORY.md for session history and current status
4. Read SECURITY.md for security status
5. Run: npm run build — verify clean build
6. Run: npm run test — verify all tests pass
7. Report:
   - Current branch and last commit
   - Build status (pass/fail)
   - Test status (X passing, Y failing)
   - Next session to work on (from MEMORY.md)
   - Any blockers or prerequisites needed
